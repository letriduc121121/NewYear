# 🎨 Hướng dẫn tách nền ảnh horse1.jpg

## 🚀 Cách nhanh nhất: Remove.bg

### Bước 1: Tách nền online
1. Truy cập: **https://www.remove.bg/**
2. Click "Upload Image"
3. Chọn file `horse1.jpg` từ `d:\NewYear\public\images\`
4. Đợi 5-10 giây
5. Click "Download" → Chọn "Download HD"
6. Lưu file với tên `horse1.png`

### Bước 2: Copy vào project
```bash
# Copy file đã tách nền vào thư mục images
Copy horse1.png → d:\NewYear\public\images\horse1.png
```

### Bước 3: Code sẽ tự động dùng PNG
- File hiện tại: `horse1.jpg` (có nền)
- File mới: `horse1.png` (không nền)
- Code sẽ ưu tiên dùng PNG nếu có

## 🎯 Lợi ích khi tách nền

### Trước (có nền):
- ❌ Nền ảnh có thể xung đột với bg2.jpg
- ❌ Khó blend với background
- ❌ Có viền cứng

### Sau (tách nền):
- ✅ Hòa quyện hoàn hảo với background
- ✅ Không có viền
- ✅ Trông chuyên nghiệp hơn
- ✅ Shadow đẹp hơn

## 🛠️ Công cụ tách nền miễn phí

### 1. Remove.bg ⭐⭐⭐⭐⭐
- **Link**: https://www.remove.bg/
- **Ưu điểm**: Nhanh, chính xác, miễn phí
- **Giới hạn**: 1 ảnh/lần, độ phân giải HD cần đăng ký

### 2. PhotoScissors ⭐⭐⭐⭐
- **Link**: https://photoscissors.com/
- **Ưu điểm**: Dễ dùng, kết quả tốt
- **Giới hạn**: Watermark nếu không trả phí

### 3. Canva Background Remover ⭐⭐⭐⭐
- **Link**: https://www.canva.com/
- **Ưu điểm**: Tích hợp nhiều công cụ
- **Giới hạn**: Cần tài khoản Pro

### 4. Adobe Express ⭐⭐⭐⭐
- **Link**: https://www.adobe.com/express/feature/image/remove-background
- **Ưu điểm**: Chất lượng cao
- **Giới hạn**: Cần tài khoản Adobe

### 5. Pixlr (Online Photoshop) ⭐⭐⭐
- **Link**: https://pixlr.com/
- **Ưu điểm**: Miễn phí hoàn toàn
- **Nhược điểm**: Phải tự select

## 📝 Sau khi tách nền xong

### Cập nhật code để dùng PNG:

Tôi sẽ tự động cập nhật code để:
1. Ưu tiên dùng `horse1.png` (nếu có)
2. Fallback về `horse1.jpg` (nếu không có PNG)
3. Tối ưu hiển thị cho ảnh trong suốt

### Bạn cần làm gì?
1. ✅ Tách nền bằng remove.bg
2. ✅ Lưu file `horse1.png` vào `public/images/`
3. ✅ Refresh trang (F5)
4. ✅ Xong!
